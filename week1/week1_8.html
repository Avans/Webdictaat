Tags voor tekstopmaak

Letterypen

Je kunt style-tags gebruiken voor het beïnvloeden van de opmaak van tekst. Voeg een style-tag toe aan een p-tag om een hele alinea te veranderen. Gebruik een span-tag met een style-attribute als je binnen een alinea even iets ander wil.

style="font-family:..." Met behulp van dit attribuut kun je het lettertype van een element aangeven.

style="font-size:..." Met behulp van dit attribuut kun je het lettergrootte in een element aangeven.

style="color:..." Met behulp van dit attribuut kun je de tekstkleur in een element aangeven.

Deze tekst staat in <span style="font-size:20pt"> en deze in <span style="font-size:small"> 

Kleur

Hier wordt een #RRGGBB-notatie voor de kleur gebruikt. Voor 255 voorgedefinieerde kleuren bestaat ook een kleuromschrijving, zoals "blue".
Zie http://www.handleidinghtml.nl/divers/kleuren/kleuren01.html voor een overzicht van kleurencode en informatie over kleurgebruik.

Lettertype
Met behulp van de style-value font-family kun je het lettertype veranderen. Het is gebruikelijk om meer dan één lettertype op te geven, gescheiden door een komma, omdat verschillende besturingsystemen verschillende lettertypen kennen (voor Windows wordt meestal Arial gebruikt, voor Unix en de Mac Helvetica). Als het lettertype niet op de machine van de browser aanwezig is, wordt meestal een standaardlettertype door de browser gekozen. Dit kan effect hebben op hoe je webpagina wordt weergegeven in de browser. Hou hier rekening mee als je afwijkende fonts gebruikt. Bijvoorbeeld:
Deze tekst staat in <span style="font-family:Courier New, monospace"> 

Font: algemeen
Je mag ook meerdere style-values samenvoegen, zoals hier:
Deze tekst staat in <span style="font-family:Courier New, monospace; color:red"> 

Overige opmaak van tekst

Behalve style-values zijn er ook speciale tags die voor een bepaalde opmaak zorgen.

Tekst vet weergeven

Hiervoor kun je de tag <b></b> (bold) gebruiken. Je kunt ook de logische tag <strong></strong> gebruiken, die meer de bedoeling van de maker dan het uiterlijk zelf weergeeft.

Tekst cursief weergeven

Hiervoor kun je de tag <i></i> (italics) gebruiken. Op vergelijkbare wijze als de strong-tag hiervoor, kun je hiervoor ook de logische tag <em></em> (emphasized = met nadruk) gebruiken.

Let op: Bij toepassen van meerdere tags op een zelfde stuk tekst, bijvoorbeeld om vet en cursief wer te even, moeten de tags correct genest worden. Bij eenvoudigere constructies, zoals vet en cursief, kan de browser fouten vaak nog wel corrigeren, maar bij complexere tags lukt dit niet altijd.

Nog wat tags om mee te experimenteren

    <u>           ondestreept
    <sub>         subscript
    <sup>         superscript
    <code>        programmacode
    <blockquote>  speciaal ingesprongen
    <pre>         getypte opmaak behouden

Speciale tekens

Om speciale tekens in HTML te kunnen weergeven maak je gebruik van de zogenaamde entiteitsnaam van dat teken.
Hoewel je sommige van deze tekens wel op het toetsenbord kunt vinden en dus rechtstreeks kunt invoeren, zal een HTML document dit teken niet (altijd) laten zien als de pagina wordt weergegeven in een browser. Om daar zeker van te zijn moet je de zogenaamde entiteitsnaam gebruiken.
Entiteitsnamen hebben de volgende vorm: &naam;
Bijvoorbeeld:

    &nbsp;   spatie                   
    &copy;   copyright-teken (©) 
    &eacute; é                 
    &euml;   ë                   
    &gt;     >                     
    &lt;     <

Zoals je kunt zien, begint iedere entiteitsnaam met een & (ampersand) en eindigt die op een ; (puntkomma).

Niet alle tekens hebben een entiteitsnaam waarmee ze aangeduid worden. Sommige tekens worden ook aangeduid met een cijfercode, bijvoorbeeld "&#177;". Met deze code kun je het plusminus teken (±) weergeven.

Zie www.handleidinghtml.nl/html/karakters/karakters01.html voor een compleet ovzicht van de speciale tekens.

Unicode

Als je netjes in Unicode en UTF werkt, kun je ook zonder veel van deze speciale tekens werken. Daarover volgt later meer.

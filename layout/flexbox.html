Flexbox Layout

LET OP! DEZE PAGINA IS NOG IN BEWERKING (Luc Claessens)

Het positioneren van containers in een pagina kan erg lastig zijn. Een eenvoudige nieuwe manier om dit netjes te doen is de flexbox layout. Voor een flexbox layout hoef je eigenlijk maar een paar dingen te doen:

  1. Je moet een omsluitende container definieren, waarin je een aantal subcontainers opneemt
  2. Je moet aangeven hoeveel ruimte elke subcontainer nodig heeft

Laten we beginnen met een eenvoudig voorbeeld. We gaan drie containers horizontaal op een rij zetten. Iedere container krijgt hierbij evenveel ruimte in de breedte.

.outer-container { display: flex; border: 1px solid black; } <div class="outer-container"> <div class="inner-container">div 1</div> <div class="inner-container">div 2</div> <div class="inner-container">div 3</div> </div>

Zie je hoe eenvoudig het is? We hoeven eigenlijk alleen voor de omvattende container flexboxing AAN te zetten.

Nu zijn alle inner-containers echter precies even breed. Dat kun je aanpassen door iedere container een relatieve flexwaarde te geven. Stel dat we de eerste en de laatste container ieder 25% van de breedte willen geven en de middelste container 50%. Je moet er dan voor zorgen dat alle inner-containers.................

.outer-container { display: flex; border: 1px solid black; } .inner-container-small { border: 1px solid blue; flex: 25; } .inner-container-large { border: 1px solid red; flex: 50; } <div class="outer-container"> <div class="inner-container-small">div 1</div> <div class="inner-container-larger">div 2</div> <div class="inner-container-small">div 3</div> </div>

Pimp my page

Nu we CSS kennen kunnen we eindelijk wat doen aan dat 1999-gevoel die HTML pagina's zonder stylesheets met zich meebrengen. De opdracht van deze week is om je eigen gemaakte pagina te pimpen met een stylesheet. In de screenshot staat een voorbeeld van een ontwerp, maar je mag ook zelf een mooi ontwerp bedenken.

Gebruik voor deze opdracht een linked stylesheet. Je moet dus zo weinig mogelijk in de HTML plaatsen, en zo veel mogelijk in het externe CSS bestand. Pas deze stijl ook toe op je gemaakte contactformulier en rooster pagina's.

Movie night

Jij wil gebruik maken van de allerlaatste CSS technieken die er maar zijn, eh? Dan is deze opdracht misschien iets voor jou, maar je moet wel heel veel zelf kunnen uitzoeken!

Maak een pagina met filmposters van je favoriete films, stijl deze mooi door de volgende elementen er in te bouwen:

  • Geef elke poster een schaduw die die op de achtergrond lijkt te werpen. Zoektermen: css, box-shadow
  • Geef de posters een perspectief effect zodat ze scheef lijken te staan. Zoektermen: css, 3d, transform, perspective
  • Zorg ervoor dat als je met je muis boven een poster hovert deze recht komt te staan en groter wordt met een animatie. Zoektermen: css, transition, transform

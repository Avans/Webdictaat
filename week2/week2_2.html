Hyperlinks

Met behulp van hyperlinks kun je je webpagina verbinden met andere webpagina's en/of verwijzen naar een bepaald onderdeel op de webpagina zelf, de zogenaamde interne koppeling.
Hyperlinks worden gemaakt met de <a>-tag, waarbij a staat voor anchor.

We onderscheiden 3 verschillende soorten hyperlinks:
  • Absolute hyperlinks;
  • Relatieve hyperlinks;
  • Interne hyperlinks.

Absolute hyperlinks

De syntax voor een absolute hyperlink ziet er als volgt uit:
<a href="url">Dit is een hyperlink</a>
De term a staat hierbij voor anchor, ofwel anker/koppeling.
De term href staat hierbij voor hypertext reference.

Een hyperlink naar de website van de Hogeschool ziet er bijvoorbeeld als volgt uit:
<a href="http://www.avans.nl/">Avans Hogeschool</a>



Relatieve hyperlinks

De syntax voor een relatieve hyperlink ziet er als volgt uit:
<a href="bestandsnaam">Dit is een hyperlink</a>
Een hyperlink naar de startpagina van je website ziet er bijvoorbeeld als volgt uit:
<a href="index.html">Startpagina</a>
De browser geeft dit weer als: Startpagina

Bevindt het bestand zich in een andere directory (map) dan het huidige bestand dan neem je in de verwijzing ook de mapnaam op. Dit ziet er dan zo uit: 
<a href="directorynaam/bestandsnaam">Dit is een hyperlink</a>
Een hyperlink naar de eerste pagina van week 2, die zich bevindt in de map week2, ziet er bijvoorbeeld als volgt uit: 
<a href="../week2/week2_1.html">Week 2</a>
De browser geeft dit weer als: Week 2



Interne hyperlinks

Voordat je interne hyperlinks kunt maken moet je de onderdelen van de pagina waar je rechtstreeks naar toe wilt kunnen springen, eerst een naam geven. Op de positie in de pagina waar je rechtstreeks naartoe wilt kunnen springen, neem je daartoe de volgende code op: 
<a name="ankernaam"></a>
Om een hyperlink naar het begin van deze pagina op nemen is direct onder de body tag de volgende code geplaatst: 
<a name="boven"></a>
Om vanaf hier direct naar het begin van de pagina te kunnen springen neem je dan onderstaande hyperlink in je document op: 
<a href="#boven"> Ga naar het begin van deze pagina.</a>
De browser geeft dit als volgt weer: Ga naar het begin van deze pagina.

Je kunt ook ankers definiëren op andere pagina's als de huidige. Dit doe je als volgt: 
<a href="andere_pagina.html#ankernaam">Ga naar anker op andere pagina.</a>
Het anker moet daar wel gedefinieerd zijn natuurlijk.


Een veel gebruikt attribuut van de anchor-tag is het target attribuut. De waarde ervan bepaalt in welk venster het bestand waarnaar verwezen wordt, wordt geopend. We komen hier bij frames nog op terug, maar nu is het alvast handig om te weten dat de waarde target="_blank" als effect heeft, dat de url geopend wordt in een nieuw venster van de browser.

Merk op, dat we in HTML-documenten de forward-slash (/) gebruiken in padnamen ipv de backward-slashes (\) die in Windows gebruikelijk zijn.

Als een gebruiker met de muis over een hyperlink beweegt, zal de cursor i.h.a. van een pijl in een handje veranderen, waarmee duidelijk wordt dat er iets 'staat te gebeuren', dit in tegenstelling tot een onderstreept stuk tekst.

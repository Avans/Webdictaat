Hyperlinks

Het web zou het web niet zijn als er geen hyperlinks waren geweest. Die blauw onderstreepte stukjes tekst zorgen ervoor dat we eindeloos blijven doorklikken naar een volgende pagina

Een link maken doe je met de <a> tag. Met het href attribuut geef je vervolgens aan waar de link naar wijst:

<a href="http://www.avans.nl/">Avans Hogeschool</a>

Deze URL die met http:// begint noemen we een absolute url. Dat is een link waar ook het domeinnaam in staat en die op elke website werkt. Als we linken naar een pagina op dezelfde site hoeven we dit niet ervoor te zetten en mag je beginnen met de slash achter het domein:

<a href="/Webdictaat/basis/lijsten.html">Ga terug naar 'Lijsten'</a> <a href="/Webdictaat">Ga terug naar homepage</a>

Relatieve URL's

Als bestanden al in dezelfde map staan hoef je helemaal niet meer het pad op te geven, je hoeft dan alleen de bestandsnaam op te geven:

<a href="lijsten.html">Ga naar 'Lijsten'</a>

Op deze manier kan je ook links maken naar bestanden die in een andere map staan:

<a href="../basis2/afbeeldingen.html">Ga naar 'Afbeeldingen'</a>

Target

Een veel gebruikt attribuut van de anchor-tag is het target attribuut. Daarmee kan je aangeven in welk venster een link wordt geladen. Als je wil dat een link altijd in een nieuw venster opent kan je dat doen met de _blank waarde:

<a href="http://bb.avans.nl/" target="_blank">Ga naar Blackboard</a>

Stylesheets

Vooraf

In de voorgaande weken hebben we tags gezien die we kunnen gebruiken om aan tekst een betekenis te geven. Zo kunnen we aangeven welke stukken tekst paragrafen vormen en wat voor soort headers daar tussen staan. De browser probeert automatisch al een toepasselijke opmaak voor te verzinnen.

<h1>In je browser ziet deze tekst er automatisch groot uit!</h1>

Nu gaan we kijken hoe we webpagina's mooier kunnen maken door een style aan elementen van de pagina toe te voegen. Op die manier kunnen we bijvoorbeeld kleuren, fonts en randen van elementen aanpassen.

In het onderstaande voorbeeld hebben we alle h1 tags van een nieuwe style voorzien!

<html> <head> <style> h1 { color: #404853; font-size: 26px; font-family: Arial; } </style> </head> <body> <h1>Hoofdstuk 1</h1> <p>Deze kopjes zien er mooier uit dan hoe de browser het standaard opmaakt.</p> <h1>Hoofdstuk 2</h1> <p>En het werkt automatisch op alle h1 tags in de hele webpagina!</p> </body> </html>

Deze style regels die we kunnen toepassen op html elementen heten Cascading Style Sheets (css). Met css kunnen we de opmaak van pagina's veranderen zonder dat we de html hoeven te veranderen.
We zeggen ook wel dat we met stylesheets de inhoud van de opmaak scheiden

Typen stylesheets

Er zijn 3 verschillende soorten manieren om stylesheets op je html toe te passen.

Linked stylesheets
In een webpagina kan een link naar een (extern) stylesheet-bestand op de webserver worden opgenomen. Zo kan hetzelfde stylesheetbestand in vele HTML-documenten gebruikt worden (hergebruik, dus consistentie en onderhoudbaarheid).
Zet bijvoorbeeld deze tag in de head om het 'mijnstylesheet.css' op de pagina toe te passen <link rel="stylesheet" type="text/css" href="mijnstylesheet.css">

Download hier een voorbeeld van een linked stylesheet

Embedded stylesheets
Kunnen binnen webpagina gedefinieerd worden met behulp van de <style>-tag. Deze tag moet binnen de head van de pagina worden opgenomen. Deze kunnen bijvoorbeeld gebruikt worden om style regels te definiëren
Syntax: <style> ... </style>
Inline stylesheets
Kan aan iedere tag binnen een webpagina worden toegevoegd door het "style"-attribuut. De style die je op deze manier meegeeft is dan alleen van toepassing op dat element
Syntax (met h1-tag als voorbeeld): <h1 style="...">...</h1>

Syntax van stylesheets

Linked en embedded stylesheets bevatten stijldefinities. Deze bestaan uit:
  • de HTML-tag die aangepast moet worden
  • een openingsaccolade ({)
  • lijst van eigenschappen/properties die aangepast moeten worden met per eigenschap/property de volgende elementen
    • naam van de eigenschap/property
    • dubbele punt (:)
    • waarde van de eigenschap/property
    • puntkomma als separator bij meerdere properties (;)
  • de sluitaccolade (})

Een voorbeelddefinitie om de h1-tag in 16-punts grootte, vet en blauw te laten afdrukken, ziet er als volgt uit:

h1 { font-size: 16pt; font-weight: bold; color: blue; }

Commentaarregels worden omgeven met /* en */, vergelijkbaar met Java en C++.

Stylesheets toepassen

Linked stylesheets

Hiervoor moet je CSS-bestand aanmaken, bv mbv NotePad/Kladblok, waarin je de stijldefinities opneemt.

Embedded stylesheets

Hiervoor wordt dus de <style>-tag gedefinieerd in het headerdeel van het huidige HTML document. De invloed van deze tag is beperkt tot het huidige document.

Inline stylesheets

Deze gebruikt u als u in een beperkt deel van een (1) webpagina een aparte opmaak wilt hanteren, b.v. tekst wilt laten inspringen. Aan de betreffende HTML-tag geef je dan het style-attribuut mee.
Voor toepassing van de stijl op een wat bredere sectie gebruik je de <span>-tag of de <div>-tag (div staat voor division), die je voorziet van een style-attribuut. Deze stijl is dan van kracht tot de eindtag </span>-tag of </div>-tag.

Zie week3_style.html voor een toepassing van deze styles!

Classes binnen stylesheets

Als je voor 1 tag (toch) meerdere stijldefinities zou willen toepassen, dan moet je volgens het bovenstaande even zo vele inline styles definieren. Met behulp van classes kan dit eenvoudiger en bovendien kun je die classes, indien gewenst, meteen voor verschillende typen tags definiëren.

Stel dat we alle belangrijke elementen rood willen maken. Dan kunnen we de belangrijke dingen de class "belangrijk" geven. Met css kunnen we deze class dan rood maken. <style> .belangrijk { color: red; } </style> <h3>Onderwerp</h3> <h3>Nog een onderwerp</h3> <h3 class="belangrijk">Belangrijk onderwerp!</h3> <h3>Laatste onderwerp</h3> <div class="belangrijk">Dit is een div. Maar wel een belangrijke div.</div>

Tools

Validatie HTML

Zie http://validator.w3.org/.
Als je je website gevalideerd hebt, mag je een plaatje als op je website plaatsen.
PS: Zie overigens ww.w3c.org voor de precieze voorwaarden!

Validatie CSS

Zie http://jigsaw.w3.org/css-validator/.
Als je je CSS-bestand(en) gevalideerd hebt, mag je een plaatje als op je website plaatsen.
PS: Zie overigens ww.w3c.org voor de precieze voorwaarden!

Link checker

Zie http://validator.w3.org/checklink.

Tidy

Tool geschreven door Dave Raggett (W3C), maar nu in onderhoud binnen de opensource community. Tidy kan o.a. case conversion, quoted attributes, en proper element nesting detecteren en corrigeren. Zie http://tidy.sourceforge.net voor een compleet overzicht van features en voor de meest recente versie voor de diverse platformen.

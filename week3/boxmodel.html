Borders, paddings & margins

We hebben nu gezien hoe we met behulp van css (Cascading Style Sheets) kleuren en lettergrootte kunnen bepalen. Nu gaan we kijken wat voor soort dingen we nog meer kunnen doen.

Border

We kunnen bijvoorbeeld een element een border geven. In onderstaand voorbeeld hebben we een div element een zwarte rand gegeven die 1 pixel dik is. Dat geven we altijd in die volgorde aan: Eerst de dikte, dan het lijnsoort en als laatste de kleur. We kunnen ook dotted of dashed als lijnsoort gebruiken. Probeer deze uit in onderstaand voorbeeld.

<style> #banaan { border: 1px solid black; background-color: yellow; } </style> <div id="banaan"> Bananen </div>

Met border geven we alle vier de randen tegelijk dezelfde stijl. Maar we kunnen ook de randen individueel een kleur geven:

<style> .vrolijk { border-left: 3px solid red; border-right: 3px solid green; border-top: 3px solid blue; border-bottom: 3px solid yellow; } </style> <div class="vrolijk"> Vrolijke kleurtjes </div>

Padding

Nu staan de randjes heel dicht om de tekst heen. Om wat meer ruimte te scheppen tussen de inhoud van de div en het randje gebruiken we padding. Daarmee geven we aan hoeveel ruimte er tussen de inhoud van de div en de rand van de div moet komen.

<style> .lekker-ruim { border: 1px solid black; padding: 30px; } </style> <h4 class="lekker-ruim"> Hier heb ik meer ruimte. </h4>

Margin

Op dezelfde manier als padding de ruimte definieert tussen de inhoud en de rand. Kunnen we met margin ruimte maken tussen de rand en de rest van de pagina.

<style> .ruimte { border: 1px solid black; margin: 40px; } </style> <h4 class="ruimte"> Hier geef ik meer ruimte. </h4>

Width & height

Standaard zijn alle elementen groot genoeg om wat er in staat goed weer te geven. Maar we kunnen ook expliciet de hoogte en de breedte aangeven in een stylesheet. Dit doen we met de width en height. Dit is de grootte zonder de padding, border en margin. Dus alleen de grootte van de inhoud

<style> #vierkantje { border: 1px solid black; width: 200px; height: 200px; } </style> <div id="vierkantje"> Ik ben een vaste grootte </div>

De manier waarop margins, paddings and borders werken in CSS noemen we ook wel het 'CSS Box Model'. Net zoals we de border aan alle vier de kanten individueel kunnen veranderen. Kunnen we dat met margin en padding ook doen, bijvoorbeeld met margin-top of padding-left.
